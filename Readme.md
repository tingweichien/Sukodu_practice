# Sudoku Solving

## Intoduce

The sukudo solver using backtracking algorithm
Building GUI by pygame and tkinter

## Requirements

- python 3.7.3
- pygame
- tkinter

## Usage

- Button.py
  - state the class for button

- Index.py
  - state the parameters

- sudoku_solving.py
  - state the algorithm to solve the sudoku

- sudoku_GUI.py
  - GUI by pygame
